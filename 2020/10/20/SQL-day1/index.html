# SQL кё°ліё к°ңл…җ

  • мҳӨлқјнҒҙ = Database software
    • мҳӨлқјнҒҙмқҖ DBл¶„м•ј м„ёкі„ м җмң  80%мқҳ лҢҖкё°м—…лӘ… мқҙкё°лҸ„н•Ё. DB SWлҘј л§Ңл“  нҡҢмӮ¬мқҙмһҗ к·ё SWмқҳ мқҙлҰ„мқҙ мҳӨлқјнҒҙ
  • DB = dataлҘј м ҖмһҘн•ҳкі  кҙҖлҰ¬н•ҳлҠ” м ҖмһҘмҶҢлҘј л§җн•Ё

# SQL = Structure Query Language (кө¬мЎ°м Ғ м§ҖлҰ¬ м–ём–ҙ)

= лҚ°мқҙн„°лҘј кІҖмғүн•ҳкі  мЎ°мһ‘н•ҳлҠ” м–ём–ҙ

  • SQLмқ„ мҷң л°°мӣҢм•ј н•ҳлҠ”к°Җ?
    • лҚ°мқҙн„°лҘј кІҖмғүн•ҳкі  лӢӨлЈ°л Өл©ҙ л¬ҙмЎ°кұҙ м•Ңм•„м•јн•Ё.

If м§„лЎңк°Җ Data Analyst / Data developer, л§ҺмқҖ м—…л¬ҙм—җм„ң SQLмқҙ л§Һмқҙ мӮ¬мҡ©лҗЁ. So л°ҳл“ңмӢң SQLмқ„ л°°мӣҢм•јн•Ё.

# SQLмқҳ мў…лҘҳ (мҙқ 5к°Җм§Җ)

1.В В В В В  Query (лҚ°мқҙн„°лҘј кІҖмғүн•ҳлҠ” м–ём–ҙ)

  • 6к°ңмқҳ м Ҳмқҙ мЎҙмһ¬
    • SELECT:В мЎ°нҡҢн•ҳкі  мӢ¶мқҖ м»¬лҹјмқ„ лӘ…мӢңн•©лӢҲлӢӨ.
    • FROM:В лҚ°мқҙн„°лҘј к°Җм ёмҳ¬ н…Ңмқҙлё”мқ„ лӘ…мӢңн•©лӢҲлӢӨ.
    • WHERE:В мЎ°кұҙм—җ л§һлҠ” лҚ°мқҙн„°л§Ң м„ нғқн•©лӢҲлӢӨ.
    • GROUP BY:В м»¬лҹј кё°мӨҖмңјлЎң лҚ°мқҙн„°лҘј к·ёлЈ№нҷ”н•©лӢҲлӢӨ.
    • HAVING:В к·ёлЈ№нҷ”лҗң лҚ°мқҙн„°м—җ лҢҖн•ң мЎ°кұҙмқ„ м„Өм •н•©лӢҲлӢӨ.
    • ORDER BY: кІ°кіјлҘј м •л ¬н•ҳлҠ” кё°мӨҖмқ„ лӘ…мӢңн•©лӢҲлӢӨ.

2.В В В В  DML л¬ё (Data Manipulation Language , лҚ°мқҙн„° мЎ°мһ‘ м–ём–ҙ)
- insert = лҚ°мқҙн„° мһ…л Ҙ м–ём–ҙ
- update = лҚ°мқҙн„° мҲҳм • м–ём–ҙ
- delete = лҚ°мқҙн„° мӮӯм ң м–ём–ҙ
- merge = лҚ°мқҙн„° мһ…л Ҙ, мҲҳм •, мӮӯм ңлҘј н•ңлІҲм—җ мҲҳн–үн•ҳлҠ” лӘ…л №м–ҙ

3.В В В В  DDL л¬ё (Data Definition Language, лҚ°мқҙн„° м •мқҳм–ҙ)
- create = мғқм„ұ
- alter = мҲҳм •
- drop = мӮӯм ң
- truncate = мӮӯм ң
- rename = мқҙлҰ„ліҖкІҪ

4.В В В В  DCL л¬ё (Data Control Language, )
- grant = лҚ°мқҙн„°лҘј м ‘к·ј н•  мҲҳ мһҲлҠ” к¶Ңн•ңмқ„ л¶Җм—¬н•ҳлҠ” кІғ
- revoke = лҚ°мқҙн„°лҘј м ‘к·ј н•  мҲҳ мһҲлҠ” к¶Ңн•ңмқ„ м·ЁмҶҢн•ҳлҠ” кІғ (ләҸлҠ”кІғ)

5.В В В В  TCL л¬ё (Transaction Control Language)
- commit = нҳ„мһ¬мқҳ лҚ°мқҙн„°мқҳ мғҒнғңлҘј DBм—җ мҳҒкө¬нһҲ м ҖмһҘн•ҳкІ лӢӨ. (лҠ” лң»)
- rollback = м§ҖкёҲк№Ңм§Җ н–ҲлҚҳ мһ‘м—…мқ„ лӢӨ м·ЁмҶҢ н•ҳкІ лӢӨ. (лҠ” лң»)
- savepoint = нҠ№м •мӢңм җк№Ңм§Җ rollback н•ҳлҠ” кё°лҠҘ

н•өмӢ¬ мқҖ лӮҙк°Җ ліҙкі  мӢ¶мқҖ лҚ°мқҙн„°лҘј м •нҷ•н•ҳкІҢ кІҖмғүн•ҙм„ң ліҙлҠ” кё°мҲ  мқ„ л°°мҡ°лҠ” кІғ. = SQL
