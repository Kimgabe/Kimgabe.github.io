# к°ңмҡ”

  • pythonмқҳ мЈјмҡ” кё°лҠҘмқё Lamda н•ЁмҲҳмқҳ к°ңл…җкіј нҷңмҡ©л°©лІ•м—җ лҢҖн•ң лӮҙмҡ©мқ„ м •лҰ¬н•©лӢҲлӢӨ.

# Lambda н•ЁмҲҳ

  • лӢЁмқјл¬ёмңјлЎң н‘ңнҳ„лҗҳлҠ” мқөлӘ…н•ЁмҲҳ

  • мқөлӘ…н•ЁмҲҳлһҖ мқҙлҰ„мқҙ м—ҶлҠ” кө¬нҳ„мІҙл§Ң мЎҙмһ¬н•ҳлҠ” к°„лӢЁн•ң н•ЁмҲҳлҘј мқҳлҜё

  • мҪ”л“ң мғҒм—җм„ң н•ңлІҲл§Ң мӮ¬мҡ©лҗҳлҠ” кё°лҠҘмқҙ мһҲмқ„ л•Ң, көімқҙ н•ЁмҲҳлЎң л§Ңл“Өм§Җ м•Ҡкі  1нҡҢм„ұмңјлЎң л§Ңл“Өм–ҙм„ң м“ё л•Ң мӮ¬мҡ©.

  • lambda н•ЁмҲҳлҘј м“°лҠ” к°ҖмһҘ нҒ° мқҙмң лҠ” мҪ”л“ңлҘј к°„кІ°н•ҳкі  к№”лҒ”н•ҳкІҢ мһ‘м„ұн•  мҲҳ мһҲкё° л•Ңл¬ёмқҙлӢӨ.

  • lambda нҢҢлқјлҜён„°(мһ…л Ҙк°’) : м¶ңл Ҙк°’ нҳ•нғңлЎң мӮ¬мҡ©

  • нҢҢлқјлҜён„° мҲҳлҠ” н•„мҡ”н•ң л§ҢнҒј м“ё мҲҳ мһҲлӢӨ.

  • returnмқҖ м“°м§Җ м•ҠлҠ”лӢӨ.

  • мқјл°ҳм ҒмңјлЎң н•ЁмҲҳлҘј л§Ңл“Өл©ҙ м•„лһҳмҷҖ к°ҷмқҙ мғқм„ұ н•ҳм—¬ мӮ¬мҡ©

  • н•ҳм§Җл§Ң мӢӨм ң мҪ”л“ңлҘј нҷңмҡ©н•ң лӢӨм–‘н•ң лҚ°мқҙн„° л¶„м„қ, AIлӘЁлҚёл§Ғ, м „мІҳлҰ¬ л“ұмқ„ мҲҳн–үн•ҳлӢӨ ліҙл©ҙ мҪ”л“ңк°Җ л§Өмҡ° кёём–ҙм§Ҳ мҲҳ л°–м—җ м—ҶмқҢ

  • мқҙ кІҪмҡ° 1лІҲмқҳ мӮ¬мҡ©мқ„ мң„н•ҙ 4~5мӨ„ лҗҳлҠ” н•ЁмҲҳлҘј мғқм„ұн•ҳлҠ” кІғліҙлӢӨлҠ”, lambda н•ЁмҲҳлҘј мӮ¬мҡ©н•ҙ к°„кІ°н•ҳкІҢ мһ‘м„ұн•ҳлҠ” кІғмқҙ мўӢмқҢ

  • мҪ”л“ңл¬ёмқҙ к°„кІ°н• мҲҳлЎқ мң м§ҖліҙмҲҳм—җ мң мҡ©н•ҳкё° л•Ңл¬ё

# мһҗмЈј м“°мқҙлҠ” labmda мҳҲмӢң top4

  • lambdaк°Җ мң мҡ©н•ҳкІҢ мӮ¬мҡ©лҗҳлҠ” 4к°Җм§Җ лҢҖн‘ңм Ғ н•ЁмҲҳ sort, filter, map, reduce

  • н•ЁмҲҳнҳ• н”„лЎңк·ёлһҳл°Қмқҳ кё°ліё мҡ”мҶҢ

  • sort : лҰ¬мҠӨнҠёлӮҙмқҳ мҡ”мҶҢл“Өмқ„ нҠ№м • мЎ°кұҙ (key=) лЎң м„Өм •н•ҳм—¬ м •л ¬

  • filter : нҠ№м • мЎ°кұҙмқ„ л§ҢмЎұн•ҳлҠ” мҡ”мҶҢл§Ң лӮЁкё°кі  н•„н„°л§Ғ

  • map : к°Ғ мӣҗмҶҢлҘј мЈјм–ҙм§„ мҲҳмӢқм—җ л”°лқј ліҖнҳ•н•ҳм—¬ мғҲлЎңмҡҙ лҰ¬мҠӨнҠёлҘј л°ҳнҷҳ

  • reduce : лҰ¬мҠӨнҠё лӮҙм—җм„ң м°ЁлЎҖлҢҖлЎң м•һ 2к°ңмқҳ мӣҗмҶҢлҘј к°Җм§Җкі  м—°мӮ°н•ҳлҠ” кё°лҠҘмқ„ н•ңлӢӨ. м—°мӮ°мқҳ кІ°кіјк°Җ лҳҗ лӢӨмқҢ м—°мӮ°мқҳ мһ…л ҘмңјлЎң м§„н–үлҗңлӢӨ. л”°лқјм„ң л§Ҳм§Җл§үк№Ңм§Җ м§„н–үлҗҳл©ҙ мөңмў… м¶ңл ҘмқҖ н•ңк°ңмқҳ к°’л§Ң лӮЁкІҢ лҗңлӢӨ.
