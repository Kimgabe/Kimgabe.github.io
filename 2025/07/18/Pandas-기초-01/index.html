# Series

  • pandas의 기본 객체 중 하나

  • numpy의 ndarray를 기반으로 인덱싱을 기능을 추가하여 1차원 배열을 나타냄

  • index를 지정하지 않을 시, 기본적으로 ndarray와 같이 0-based 인덱스 생성, 지정할 경우 명시적으로 지정된 index를 사용

  • 같은 타입의 0개 이상의 데이터를 가질 수 있음

  • 실제 분석에서 생성하는 경우는 거의 없다.

  • DataFrame에서 파생되는 경우는 많음
