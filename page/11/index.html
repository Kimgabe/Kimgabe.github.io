# Numpy

Numpy를 사용하는 이유

  1. 파이썬 list보다 뛰어난 성능 (작업 처리 속도가 빠르다)

  2. 메모리 사이즈가 작다 (파이썬 리스트보다 적은 메모리로 처리 가능하다)

  3. 유용한 Built-in function (선형대수, 통계 관련 함수 등)

  • numpy에서 사용되는 다차원 리스트를 표현할 때 사용되는 데이터 타입
