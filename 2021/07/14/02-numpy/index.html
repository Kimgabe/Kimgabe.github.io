# seed 함수

  • 위에서 실습한 random함수는 실행할때마다 매번 값이 바뀌게 된다.

  • 랜덤한 값을 동일하게 다시 생성하고자 할때 사용

# choice

  • 주어진 1차원 ndarray로 부터 랜덤으로 샘플링

  • 정수가 주어진 경우, np.arange(해당숫자)로 간주

[19, 33, 59, 39],
[15, 36, 65, 1]])

[1.5, 2.6]])
