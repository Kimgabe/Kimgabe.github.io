# cusum (누적 합계)

  • 0번째는 기존 y의 0번째 값, 2번째부터는 0번째 + 1번째 원소의 값, 3번째는 누적된 합계 + 3번째 원소의 값...

  • 와 같이 누적된 합이 출력된다.

  • 그래프 등을 그릴때 활용하면 유용하다.
